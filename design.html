Design Process


When designing our base robot, we established criteria first. We considered things such as the footprint, height, balance, drivetrain, sensors to be used, user interface, etc. to arrive at an optimal base robot. We also strived to design and build a robust base robot, which features a frame around the chassis to provide structural rigidity as well as means to square up against walls and mission models. When we design attachments, we keep in mind that these attachments need to be robust, easy to install and remove, and both passive and multi-purpose whenever possible.


Our Base Robot


Our base robot is compact with a strong frame to help with consistently accurate driving. The robot’s drive wheels are narrow with a medium diameter, which provides a balance between speed and precision. We used the hub’s built-in gyro sensor, to allow the robot to turn and move precisely. The initial design included 2 colour sensors for line-following and squaring up on the game mat. However, we found that we did not require the colour sensors as gyro driving and physical squaring sufficed for accuracy.
